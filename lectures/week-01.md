# 第 1 週：網頁設計概論與基礎

## 課程目標
1. 了解網頁內容的基本元件及應用方式。
2. 具備應用 HTML 設計網頁元件的技能。
3. 能夠以 CSS 設計各種網頁元件的外觀。
4. 具備閱讀 HTML + CSS 程式碼的能力。

## 需要軟體
* **記事本** (甚至可以不需要)。
* **VS Code** (建議安裝)。
* **命令列 / 終端機** (Windows: `cmd`, Mac: `terminal`)。

## 永遠的第一個 Hello World!
在終端機輸入以下指令：
```bash
echo Hello World > index.html
```
這會產生一個 `index.html` 檔案，雙擊即可在瀏覽器開啟。

## 網頁設計核心：UI x UX
* **UI (User Interface)**：外觀 (設計出什麼)。
* **UX (User Experience)**：體驗 (為什麼這麼設計)。

## HTML 與 CSS 的關係
| 項目 | 功能 |
| :--- | :--- |
| **HTML** | 內容文本 (Structure) |
| **CSS** | 外觀樣式 (Presentation) |

**好處：** 實現「文本與樣式分離」。

---

## HTML 基礎標籤

### 結構
* `<html>`：網頁所有內容。
* `<head>`：機器可讀資訊 (Metadata)、CSS 載入、Open Graph 設定。
* `<body>`：瀏覽者看到的實際內容。

### 常見標籤
* **標題**：`<h1>` 到 `<h6>` (每頁只能有一個 `<h1>`)。
* **段落**：`<p>`。
* **超連結**：`<a href="URL" target="_blank">`。
* **圖片**：`<img src="image.jpg" alt="替代文字">`。
* **水平線**：`<hr>`。
* **換行**：`<br>`。
* **區塊容器**：`<div>`。
* **行內容容器**：`<span>`。

### 標籤類型
1. **成對標籤**：`<開始標籤> 內容 </結束標籤>`。
2. **空元素**：單一存在，如 `<br>`, `<hr>`, `<img>`, `<input>`。
