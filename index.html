113-2 網頁設計(甲)

世新大學資訊傳播學系 | 授課教師：吳林展

本課程教材以 Honkit（Gitbook 的開源分支）建置，內容涵蓋網頁設計基礎知識。

課程目標

  1. 了解網頁內容的基本元件及應用方式
  2. 具備應用 HTML 設計網頁元件的技能
  3. 能夠以 CSS 設計各種網頁元件的外觀
  4. 具備閱讀 HTML + CSS code 的能力
