## 推薦書籍
* Akiko Hayashi, Hiroshi Hara, Junya Yamada, Kumiko Hiramoto (2014)。**這個設計好在哪? 看懂平面, 網頁及廣告設計的奧秘：設計人一定要答對的 55 個問題**。旗標。
* Akiko Hayashi, Hiroshi Hara, Junya Yamada, Kumiko Hiramoto (2016)。**這個設計好在哪？2：看懂平面，網頁及廣告設計的奧秘 - 做設計前，請先答對這 46 個問題**。旗標。
* Tracy Osborn (2021)。**如何設計好網站之 UX 與美學基礎**。碁峯。
* Steve Krug (2006)。**如何設計好網站：Don't Make Me Think**。上奇。
* エ・ビスコム・テック・ラボ (2017)。**設計師都該懂的 UI/UX 設計實務：超圖解跨裝置網頁設計實戰講座**。博碩。
* 池谷義紀 (2008)。**Web+設計的黃金則：讓人流連忘返的網站設計關鍵**。旗標。
* 李建杭 (2020)。**金魚都能懂的 CSS 選取器：金魚都能懂了你還怕學不會嗎**。博碩。
* 李建杭 (2022)。**金魚都能懂的 CSS 必學屬性：網頁設計必備寶典**。博碩。
* 羅蘋‧威廉斯著 (2021)。**好設計，4 個法則就夠了：頂尖設計師教你學平面設計，一次精通字型、色彩、版面編排的超實用原則**。臉譜。
