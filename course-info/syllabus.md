# 113-2 網頁設計 課程介紹

## 教學目標
網頁是透過網路進行資訊傳播的基本方式，合適的網頁設計與製作能夠提供使用者有效而愉悅的網路資源使用環境。因此，這門課程是資訊傳播學系的核心知能之一，並且和其他的課程，如互動程式設計、標示語言、網站規劃與製作都有密切的關係。

為了能夠學習更為專業的網頁設計知能，本課程將有別於一般的網頁設計與製作課程，我們將不會以編輯軟體為中心的技巧作為教學的目標。而是希望以網頁的整體設計為對象，希望同學對整個網頁的內容設計有全面的認識。並且配合 HTML5 及 CSS3 的方式來設計這些網頁中的元件。

本課程的具體目標如下：
1. 了解網頁內容的基本元件及應用方式。
2. 具備應用 HTML 設計網頁元件的技能。
3. 能夠以 CSS 設計各種網頁元件的外觀。
4. 具備閱讀 HTML + CSS code 的能力。

## 授課方式
講授、實作
